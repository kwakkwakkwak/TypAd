#  MD을 지원하는 메모장 

## 메모장

**Goals**

1. 메모장을 개발한다.
2. MD 파일 편집을 가능하게 한다.
3. JS기반 어플리케이션을 연습한다.
4. 추가적으로 ELECTRON + React 를 공부하고 연습한다.
5. totoro 스터디

**기능정리**

```
- 메모장
	- 기본적인 메모장 기능 구현
		- 새로만들기
		- 저장
		- 불러오기
		- 편집
	- 추가적인 메모장 기능
		- MD 지원
		- 자동 들여쓰기 기능
		- Short cut 커스터마이징
		- 맞춤법, 문법검사
		- 찾기, 찾아바꾸기 기능
```
